# ReadMe
### 何？
* 株価指数と、政府統計（日本銀行API）を1ページで閲覧・比較するチャート。
* 上段＝株価指数チャート、下段＝政府統計（日銀API）チャート。
* `data/` 配下のCSVを `index.html` で読み込んで表示する。
* 時間軸（日次／週次／月次）は全チャート連動で切り替わる。

### 株価指数（上段チャート）
* `list.csv` に列挙した指数名ごとに、`data/<指数名>.csv`（四本値CSV）の終値を読み込んで表示する。
* 指数名と `data/` 配下のファイル名が不規則な場合は `index.html` 内の `IDX_DATA_ALIASES` で対応付けする。
* 複数選択時は起点100で正規化した上昇率比較も併記する。

### 政府統計（日銀API・下段チャート）
* `update_boj.js` が日本銀行「時系列統計データ検索サイト」APIから政府統計（為替・金利・マネー・短観・物価等）を取得する。
* 階層API(getDataLayer)でDBを丸ごと取得し、`data/boj/<DB>_<期種>.csv`（ワイド形式）と `data/boj/_catalog.csv` を出力する。
* `bojlist.csv` でチャートのプルダウンに出す系列を厳選できる（空なら _catalog.csv の全系列を自動表示）。列に集約方法(last=期末値 / sum=合計)を指定可能。
* APIキー不要。クレジット表示はindex.html下部に記載。高頻度アクセス禁止のためリクエスト間隔を空けている。
* `.github/workflows/update_boj.yml` で取得（当面は手動実行のみ。安定後に毎週金曜20:00の自動化を有効化予定）。

### 時間軸の粒度切替
* 画面上部の「日次／週次／月次」ボタンで全チャートの時間軸を連動切替する。
* 集約ルール：水準系（株価・為替・金利等）＝期末値。政府統計は系列ごとに集約方法を上書き可能。
